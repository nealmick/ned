Key Bindings info page.
CTRL and CMD are used interchangably depending on mac/windows keyboard layout

--finder
CMD F 
	- Open finder window
	- Enter/return - find next
	- shift enter/return - find previous

--copy paste
CMD V
	- Paste current clipboard content to cursor

CMD C
	- Copy cursor selection

CMD A
	- Select full file


--cursor 
CMD LEFT
	- Move cursor to line start

CMD RIGHT
	- Move cursor to line end

CMD UP
	- Move cursor up 6 lines

CMD DOWN
	- Move cursor down 6 lines

CMD W
	- move cursor to next word

CMD SHIFT W
	- move curosr to last word

CMD : or CMD ;
	- Enter line number
	- Jump to line number

--undo redo
CMD Z
	- Undo last change

CMD SHIFT Z
	- Redo last change

--fontsize
CMD +
	- increase font size

CMD 0
	- decrease font size

--bookmarks
CMD b
	- show/hide bookmarks window

CMD 1-9
    - jump bookmarks 1-9

CMD shift 1-9
    - set bookmark at cursor

--Settings
CMD ,
	- Launch settings editor
